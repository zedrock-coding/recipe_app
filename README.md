# recipe_app

A flutter app to store and manage recipes with ease and also simple UI for users.

Current Version: 1.0.2

## Getting Started

This app provides the functionality to the user, to add recipes, including all their methods, ingredients, servings, etc.
The app is modest and simple for people to use.

## New Features (v1.0.2)

- Added functionality to add new recipes.
- Implemented recipe list with expandable details.
- Fixed UI typos and updated documentation.
